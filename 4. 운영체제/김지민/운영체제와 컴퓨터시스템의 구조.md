# 운영체제와 컴퓨터시스템의 구조
운영체제는 앞단의 어떤 인터페이스를 두느냐에 따라 GUI와 CUI로 나눌 수 있음

#### GUI(Graphical User Interface)란?
그래픽을 사용하여 컴퓨터와 상호작용하는 인터페이스
ex) windowOS, macOS 등 현대의 OS가 이를 대표


#### CUI(Character User Interface)란?
사용자가 키보드만을 사용하여 문자를 기반으로 컴퓨터와 상호작용하는 인터페이스
ex) 예전 OS인 MS-DOS가 대표(1994년 단종)

#### 운영체제의 역할
- CPU 스케줄링과 프로세스 상태관리
- 메모리 관리
- 디스크 파일 관리
- I/O 디바이스 관리

#### 운영체제의 구조
굵은 글씨 부분이 OS
- 유저 프로그램
**- 인터페이스(GUI or CUI)
- 시스템콜(system call)
- 커널(I/O 드라이버, 파일시스템 등)**
- 하드웨어(hardware)

#### 컴퓨터 시스템의 구조
- CPU : 인터럽트에 의해 메모리에 존재하는 명령어를 해석해서 실행하는 일꾼
- DMA컨트롤러 : CPU의 일을 보조하는 일꾼
- 메모리 : 전자회로에서 데이터, 상태 등을 기록하는 장치(작업장)
- 타이머 : 특정 프로그램에 시간을 다는 역할
- 디바이스 컨트롤러 : IO디바이스들의 작은 CPU
- 로컬버퍼 : 디바이스에 달려 있는 작은 메모리

#### CPU
산술논리연산장치, 제어장치, 레지스터로 구성되어 있는 장치. 인터럽트에 의해 메모리에 존재하는 명령어를 해석해서 실행하는 일꾼.

- 산술논리연산장치
![](https://velog.velcdn.com/images/shypanda0119/post/a7ff8519-ed0b-47af-b22b-4d82ac2dac7c/image.png)
ALU(Arithmetic and Logical Unit, ALU)는 덧셈, 뺄셈, 곱셈, 나눗셈 등 산술연산과 논리연산을 하는 회로장치를 말합니다.

- 제어장치
제어 장치(Control unit, CU)는 프로세스의 조작을 지시하며 명령어들을 읽고 해석하며 데이터 처리를 위한 순서를 결정

- 레지스터
CPU 안에 있는 매우 빠른 임시기억장치

- 캐시메모리
