# 유선 LAN과 무선 LAN

## 유선 LAN
유선 LAN은 **IEEE 802.3** 표준 규격을 따르며, 현대 이더넷 환경은 대부분 **전이중화 통신**을 기반으로 구축된다.

### 통신 방식
* **전이중화 통신(Full Duplex)**: 송신로와 수신로가 분리되어 있어 양쪽 장치가 **동시에 송수신**할 수 있는 방식. 현대 이더넷의 표준이다.
* **CSMA/CD**: 과거 반이중화 방식에서 사용하던 충돌 감지 기술.
    * 회선 사용 여부 확인 → 비어있으면 전송 → 충돌 발생 시 일정 시간 대기 후 재전송

### 전송 매체 (케이블)
1. **트위스트 페어 케이블**: 구리선을 꼬아서 만든 케이블
    * **UTP**: 실드 처리를 하지 않은 일반적인 LAN 케이블
    * **STP**: 실드 처리를 하여 외부 노이즈에 강함
    * **커넥터**: 흔히 사용되는 **RJ45** 커넥터를 사용
2. **광섬유 케이블**: 레이저를 이용해 빛의 속도로 데이터를 전송 (100Gbps+)
    * **원리**: 굴절률이 높은 **코어(Core)**와 낮은 **클래딩(Cladding)**을 이용해 빛을 전반사시켜 전송
    * **특징**: 장거리 전송에 유리하며 전자기 간섭이 없음


## 무선 LAN
무선 LAN은 **IEEE 802.11** 표준(Wi-Fi) 규격을 따르며, 공기라는 매체를 공유하므로 **반이중화 통신**을 사용한다.

### 통신 방식: CSMA/CA
무선 환경은 충돌 감지가 어려워, 충돌을 사전에 **회피**하는 방식을 사용.

1. **채널 감지**: 채널이 사용 중인지 확인
2. **IFS(InterFrame Space)**: 우선순위에 따라 일정 시간 대기
3. **랜덤 백오프**: 랜덤 시간만큼 추가 대기 후 프레임을 전송
4. **ACK 확인**: 제대로 송신되었다면 ACK 세그먼트를 받으며 종료하고, 받지 못하면 재시도



### 주파수 대역 2.4GHz vs 5GHz
|  | 2.4GHz | 5GHz |
| :--- | :--- | :--- |
| **속도** | 상대적으로 느림 | **매우 빠름** |
| **장애물 투과** | **강함** (벽 통과 유리) | 약함 (감쇄 심함) |
| **커버리지** | 넓음 | 좁음 |
| **호환성** | 높음 (구형 기기 지원) | 상대적으로 낮음 |
